# JEE Topic Importance & Weightage Guide

## How to Use This File

Each topic is rated on two axes:
- **JEE Mains Weightage** — approximate % of marks historically (2019–2024 trend)
- **JEE Advanced Weightage** — frequency + conceptual depth required

**Priority Tiers:**
- 🔴 **CRITICAL** — Very high marks yield; never skip
- 🟠 **HIGH** — Consistently asked; study deeply
- 🟡 **MEDIUM** — Asked in ~50% of years; worth covering
- 🟢 **LOW** — Occasionally asked; do after the above are solid

## Strategic Study Order (Based on ROI)

### Phase 1 — High Yield Foundation (do first)
**Physics:** Mechanics (NLM, Work-Energy, Rotation) → Current Electricity → Electrostatics
**Chemistry:** Mole Concept → Chemical Bonding → Thermodynamics (chem) → GOC → Carbonyl Compounds
**Math:** Calculus (Limits → Diff → Integration → AOD) → Coordinate Geometry (Circles → Straight Lines)

### Phase 2 — Core JEE Topics
**Physics:** EMI → Waves → Ray Optics → SHM → Modern Physics
**Chemistry:** Electrochemistry → Kinetics → Ionic Equilibrium → p-Block → Coordination
**Math:** Complex Numbers → Matrices → Vectors & 3D → Probability → AP/GP

### Phase 3 — Complete Coverage
**Physics:** Alternating Current → Nuclear → Fluid Mechanics → Thermodynamics (phys)
**Chemistry:** Stereo → Amines → Alcohols → s-Block → Solid State → Surface Chem
**Math:** Conic Sections (Parabola, Ellipse, Hyperbola) → Trigonometry → Binomial → P&C

### Phase 4 — Lower Priority (if time allows)
**Physics:** Semiconductor devices
**Chemistry:** Environmental Chemistry → Polymers → Biomolecules
**Math:** Mathematical Reasoning → Induction → Statistics

## JEE Mains vs Advanced: Key Differences

| Aspect | JEE Mains | JEE Advanced |
|---|---|---|
| Format | 30 Qs per subject (MCQ + numerical) | Multiple types: MCQ, MRQ, integer, matrix |
| Difficulty | Moderate; formula-heavy | Concept-heavy; multi-step reasoning |
| Trick Topics | Semiconductor, Statistics, Biomolecules (scoring) | Rotation, Electrostatics, Organic mechanisms |
| Time pressure | High (3 hrs for 90 Qs) | Very high (3 hrs × 2 papers) |
| Best strategy | Speed + accuracy on standard problems | Deep conceptual understanding |
